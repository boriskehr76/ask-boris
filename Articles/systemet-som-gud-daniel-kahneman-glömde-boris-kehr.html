Systemet som Gud (Daniel Kahneman) glömde

Created on 2022-03-30 14:37

Published on 2022-03-30 14:55

Jag uppskattar verkligen Susan Weinschenks nya kurs i beteendedesign på Ambition Empower. Det var första lektionen var en introduktion till hur vi fattar våra beslut. Är man bekant med Daniels Kahnemas bok, Tänka snabbt och långsamt så känner man konceptet med de två tänkesätten. System 1, det snabba, energieffektiva sättet som bygger på vanor, impulser och omedvetna beslut. System 2, det långsamma och analytiska som kräver tid och energi och reserveras för svåra eller viktiga beslut. Vi använder det ena eller det andra. Antingen går vi på rutin eller så stannar vi upp och tänker efter. Både sätten är väldigt bra och kompletterar varandra utmärkt. Det enda negativa med konceptet är den förvirrande namngivningen. Funkade inte snabb och långsam som på boktiteln? Nu måste jag tänka långsamt varje gång jag ska komma ihåg vilket system som var vilket. 

Vad menar jag med system 3? Om system 1 är våra vanor och system 2 vårt logiska tänkande så är system 3 den mänskliga faktorn där vår personlighet, känslor och ego får sista ordet. System 3 är där system 1 och system 2 möts, både matematiskt och konceptuellt. System 3 kickar in där logiken inte längre räcker till, som vilken partner du ska välja eller bestämma vilken kattunge som är sötast. Det är genomtänkta val baserade på magkänslan och inte logiken. 

Enligt Susan använder vi antingen det ena eller det andra systemet när vi gör ett val. En produkt bör därför vara designad efter vilket system som kommer att användas mest. Till exempel en chipspåse ska locka till spontanköp och är knappast ett viktigt beslut. Typisk system 1. Designen behöver vara uttrycksfull, lättbegriplig och tala direkt till kunden. Ska vi däremot designa för val av pensionsfond kommer prioriteringarna vara helt annorlunda. Här måste vi skapa trygghet och förtroende samt tillhandahålla en massa information innan användaren kan göra ett val. 

För system 3 tar vi alla känslor från system 1, och låta dem köra över all logik i system 2. Ett perfekt exempel är den första iPhonen. När den presenterades var den underlägsen sina konkurrenter på nästan alla punkter. Den saknade fysiskt tangentbord, hade dålig täckning, usel batteritid och bara ett par förinstallerade appar. Ändå fick den kunderna se förbi alla brister och längta efter att svepa fingrarna över den (på den tiden) gigantiska skärmen. 

Alla lyxvaror kräver ett större (ekonomiskt) beslut men säljaren vill samtidigt inte att du ska börja tänka för mycket. Det är därför de smyger förbi ditt sunda förnuft (till exempel genom att inte ens visa eller nämna priset), och ger sig direkt på ditt ego. Varför ska du köpa denna extremt dyra klocka eller denna löjligt bensintörstiga sportbilen? För att du är värt det. För att du äntligen kan vara dig själv. För att alla andra ska se dig. För att ingen nobelpristagare ska komma och säga åt dig hur du ska och inte ska tänka. Snabbt eller långsamt? Jag tänker som jag själv vill. Så de så.
